# Vue3.0+vue-router+vuex 开发去哪儿网 WebApp

```
// git clone
git clone https://github.com/oOBobbyOo/vue3.0-qunar.git

// Project setup
npm install

// Compiles and hot-reloads for development
npm run serve

// Compiles and minifies for production
npm run build
```
